URL Problems
1.1   URLs that are Newly Broken/Redirecting/No More Content
For any URL that use to work, but is now broken or redirecting, it should show up here. If no content has been imported in a month, it should also show up here... often times, bloggers put up a post which updates users about their new URL... that will need to be copied manually.

If we update the URL, the Update URL button needs to be clicked. If no change is needed... which would really only happen if content has stopped for no reason, we click NO Problem, which will reset the clock... and we'll check again in 30 days if content still has not been updated. If the URL is legitamately broken, then we Blacklist it. At which point, it needs to be removed from Search. If there has been content, and the URL is incorrectly showing up on this chart, A Bug needs to be Submitted.
