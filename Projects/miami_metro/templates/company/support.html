Overview of Shopping Features

Deal Aggregation

We pull all running promos into a filterable dashboard of color-coded promos.

Historical Prices

We apply these promotions to the items a user is watching in order to tell them the lowest price at any point in time as well as provide them with historical price tracking charts so that users can set up smart price alerts.

Price Alerts

Users can set up price alerts on items they like. Then when the price drops, we'll let them know.

Size Availability

We keep up with the availibility of a user's size and color preferences... so that they're not keeping up with items that have long since been in stock.

How To Shelf an Item While Shopping

Select Size

Whenever you're shopping, and you see something you like... you can bookmark that item using our bookmarklet. If you care about a particular size... you can select YOUR SIZE before you click the bookmarklet... And then we'll keep track of the availibility for you. (You can edit the size later, if you forget to do it here.)

Click Bookmarklet

To add an item to your Shelves, just click the Shelf-it Bookmarklet (that you installed using the Bookmarklet Section above). Remember to select your size beforehand if you want to track the availibilty for your size.

Confirmation

After you click the bookmarklet, you'll see a confirmation popup. Before click Save, you can configure a few things.

Settings & Save

If you want to go ahead and set up a price alert (so that you get notified when the item goes on sale, you can do that here. You should also choose a Shelf for this item to show up on. (If you don't select a Shelf, you can still find it in the My Favorites Shelf.) Then make sure to click SAVE!
